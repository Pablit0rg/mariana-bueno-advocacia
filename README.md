# Mariana Bueno - Advocacia de Impacto

Landing Page desenvolvida para advogada especialista em Direito Civil e Trabalhista.
O projeto foca em transmitir autoridade ("Mente de Procuradora") através de uma estética "Dark & Gold" e performance otimizada para conversão.

## Tecnologias
- HTML5 Semântico
- CSS3 (Variáveis, Flexbox, Grid)
- JavaScript (Vanilla)
- Mobile-First Design

## Objetivo
Converter visitantes em agendamentos via WhatsApp, focando em casos de alta complexidade (Justa Causa, Inventário).

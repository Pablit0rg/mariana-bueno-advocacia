# Diário de Bordo - Mariana Bueno Advocacia

**Status:** INICIALIZAÇÃO (v1)
**Foco:** Conversão, Autoridade, Estética "Black Excellence" (Dark & Gold).

## Objetivo
Criar uma Landing Page de alta performance para advogada (Mariana Bueno), focada em captar clientes de casos trabalhistas e cíveis complexos.

## Stack & Workflow
* **Core:** HTML5 Semântico + CSS3 Moderno + JS Puro (Vanilla).
* **Design System:** "Dark Mode" nativo. Fundo escuro (Rich Black), tipografia serifada (Cinzel) para autoridade, acentos dourados.
* **Mobile-First:** Todo o CSS é escrito primeiro para telas pequenas, depois expandido para Desktop.
* **Acessibilidade:** Foco visível, tags ARIA e contraste alto.
* **Estrutura de Navegação:** Landing Page única com 4 âncoras (Atuação, A Advogada, Depoimentos, Contato).
